The legal bases of data privacy are defined in the German Federal Data Protection Act (Bundesdatenschutzgesetz) and the German Telemedia Act (Telemediengesetz).

Access Data / Server Log Files

The service provider (respectively his web space provider) collects data on every access to the offer (so called server log files). These data include:

Name of the website requested, file name, date and time of the request, amount of data transferred, notification about successful request, browser type including version and operating system, referrer url (the website visited previously) IP address and the requesting provider.

The service provider uses log files only for the purpose of operating, securing and optimizing the offer. However, the service provider reserves the right to inspect the log files later on if there is reasonable suspicion of unlawful use due to concrete indications.

Handling of Personal Data

Personal information is information that will help to identify a person, i.e. data that can be traced back to a person. This includes the name, email address or telephone number. However, data on preferences, hobbies, memberships or which web pages were viewed by someone are counted among personal data, too.

Personal data is only collected, used and passed on by the service provider if permitted by law or the user's consent to the data collection.

Contact

When contacting the service provider (for example by contact form or by e-mail), the details of the user will be saved in order to process the request and for the case that follow-up questions arise.

Integration of Services and Contents of Third-Parties

It can happen that within this online offer, third party content is integrated from other websites, such as YouTube videos, maps from Google Maps, RSS feeds or graphics. This always implies that the provider of such content (hereinafter referred to as "third-party") notices the IP address of the user. Without the IP address, they could not send the content to the browser of the user. The IP address is required in order to present this content. We strive to use only such content where their respective providers are using the IP address only to deliver the content. However, we have no control if the third-party saves the IP address such as for statistical purposes. As far as known to us, we will clarify the users about it.

Cookies

Cookies are small files that allow to store specific information related to the access device of the user (computer, smartphone etc.) on that device. On the one hand, they improve the usability of websites and therefore the users experience (for example storage of login data). On the other hand, they help to collect statistical data about the website utilisation which may be analysed later on to improve the offer further. The user can influence the usage of cookies. Most browsers are equipped with an option to restrict or completely prevent the storage of cookies. However, the functionality and in particular the ease of use may be limited without cookies.

You can manage cookies of many online advertisement services via the US website http://www.aboutads.info/choices/ or via the EU website http://www.youronlinechoices.com/uk/your-ad-choices/ .

Piwik

This offer uses Piwik , an Open-Source software for statistical analysis of user access. Piwik uses “Cookies”, which are text files that are saved on the computer of the user and allow the analysis of how the users use the website. The information gained from the cookie about the use of the offer will be stored on the service provider's servers in Germany. The IP address is anonymized immediately after processing and before storage. Users may refuse the use of cookies by selecting the appropriate settings in their browsers. However, the service provider indicates that in this case, the user may not be able to use all features of this website.

Cancellation, Changes, Corrections and Updates

The user has the right to obtain information about the personal data saved about him free of charge. In addition, the user has the right to correct inaccurate data, block or delete his personal data, as long as there is no legal obligation to retain this data.
